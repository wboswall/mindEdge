<h4>CSS Background </h4>
<p><a role="button" tabindex="0" class="glossary-alert" id="glossTerm0" data-term="Cascading Style Sheets" name="Cascading Style Sheets" title="A style sheet language used for describing the presentation of a document written in a markup language like HTML.">Cascading Style Sheets<span class="screenreader">*</span></a><span style="display:none;">&nbsp;A style sheet language used for describing the presentation of a document written in a markup language like HTML.&nbsp;</span> (CSS) is a World Wide Web Consortium (W3C) standard that defines how documents written in <a role="button" tabindex="0" class="glossary-alert" id="glossTerm3" data-term="HTML" name="HTML" title="Hypertext Markup Language is a standard Web protocol that allows web pages to be displayed on the Web.">HTML<span class="screenreader">*</span></a><span style="display:none;">&nbsp;Hypertext Markup Language is a standard Web protocol that allows web pages to be displayed on the Web.&nbsp;</span> should appear on a webpage. The appearance of web elements includes such items as color, background, dimensions, formatting text, position, and the behavior of those elements.</p>
<p>Click on the following tab to learn more about the evolution of CSS.</p>
  
  <dl class="jquery_accordion">
<dt>Evolution of CSS</dt>
	<dd><p>The CSS file type was originally proposed in 1996 by Dr. H&aring;kon Wium Lie, a Norwegian web developer. That first version was called the CSS Level 1 Recommendation or CSS1. Although the first version of CSS had features for setting and displaying page style elements, there was a lack of dependable browser support at that time. As a result, CSS was not adopted to a significant degree for many years.</p>
	  
<p>In 1998, CSS Level 2 (CSS2) was released, featuring page layout properties and styles for other media types. Then, CSS Level 2, Revision 1 (CSS2.1) was released with minor adjustments to CSS2.</p>
	  
<p>In contrast to earlier versions, CSS Level 3 (CSS3) is divided into individual modules. Each module addresses a separate feature, whether it is borders, animation, or multiple column layouts, for example. Some modules are being standardized and released one at a time while others will be experimental. This allows developers to start using one feature at a time instead of having to wait for a complete specification to be &quot;ready.&quot; Updates and additional information are available on the W3C website: <a href="https://www.w3.org/" target="_blank">https://www.w3.org/</a>.</p>
