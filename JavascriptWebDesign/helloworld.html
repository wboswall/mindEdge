Unlike external css that must be in the head or else the html would render but then immediately the site would be styled. That is very shocking to the visitor.

However unlike external CSS, Javascript can either be in the head or just above the closed body tag. This way if you have website that heavenly relies on the javascript then put it in the head tag. Having said that, if you have alot of onload, mouseover, and little scripts for features then it is best to put that above the closed body tag. This way the site loads quickly and it gives the viewers a chance to start reading the website while the not-so-important javascript is loading.

However if you have a script that has both important and reliant js and unimportant js. It would then be a good idea to separate them and put the important js at the top and the unimportant just above the closing body tag.
