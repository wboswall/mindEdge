1. True or False?

HTML5 introduced new tags that allow users to embed multimedia elements directly into the page without having any external support.

a. True

Correct. This is a true statement. HTML5 introduced new tags introduced that deal with multimedia, for example, <audio>, <canvas> and <video> that allow users to embed these multimedia elements directly within a webpage.

b. False

Incorrect. Try again.

2. True or False?

HTML5 introduced several new semantic tags.

a. True

Correct. This is a true statement. HTML5 introduced new semantic tags, for example new structural element tags such as <section>. to provide improved semantic support for webpage structure.

b. False

Incorrect. Try again.
